# Getting the audio show file up and running

The Main Room console keeps the venue's standard show file as the starting point for every event. Always start from the house show file rather than a blank scene.

## Loading the show file

1. Power the console via the sequenced mains — never flick the rack power directly.
2. Wait for the console to finish booting, then open the file browser on the console surface.
3. Load the most recent dated copy of **ILUMINA HOUSE SHOW** from the console's internal drive (a backup lives on the USB stick taped inside the console lid).
4. Recall the scene named **BASELINE** — this restores the standard patch, output routing, and house EQ.

## After loading

1. Check the patch page matches the day's input list; save a new copy under the event name before making changes (**never overwrite the house file**).
2. Line-check every input with the stage tech before doors.
3. Confirm the matrix outputs to the PFA and back-of-house zones are unmuted at show levels.

## Safety note

> Amplifier racks and mains distribution must only be powered via the sequenced power controller. If the sequencer has faulted, stop and call the head technician — do not bypass it.

Verify this procedure against this page before working on mains power.
